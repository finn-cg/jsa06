What is HTML?

  2. HTML is the standard markup language for creating Web pages

  3. HTML describes the structure of a Web page

  4. HTML consists of a series of elements

  5. HTML elements tell the browser how to display the content

  6. HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc

What is HyperText?

Hypertext is text displayed on a computer display or other electronic devices with references (hyperlinks) to other text that the reader can immediately access. Hypertext documents are interconnected by hyperlinks, which are typically activated by a mouse click, keypress set, or screen touch. Apart from text, the term "hypertext" is also sometimes used to describe tables, images, and other presentational content formats with integrated hyperlinks. Hypertext is one of the key underlying concepts of the World Wide Web, where Web pages are often written in the Hypertext Markup Language (HTML). As implemented on the Web, hypertext enables the easy-to-use publication of information over the Internet.

What is a Language?

HTML elements tell the browser how to display content. At the same time, they label pieces of content like “here is a title”, “here is a paragraph”, “here is a link”, etc.

Structure of an HTML element:

Opening tag: This tag includes the name of the element, enclosed in curly braces "<title>". This tag marks where the element begins or has an effect.

Content: The part is written after the opening tag, in this example, the content is a piece of text.

Closing tag: This tag is wrapped in curly braces "</title>" This tag has the effect of marking where the element ends or where the element expires.

However, there are still a few tags that only have an opening tag without a closing tag.

What is right and wrong syntax?

Most opening tags must be closed by itself (closeing tag).

Tags without any content still have to have both closing and opening tags, the traditional opening and closing style is no longer useful at present.

A few other principles

Each tag has its own attributes, and those attributes are separated by spaces.

Attribute values ​​can be set in double quotes or single quotes.

In some cases their values ​​can include both double quotes and single quotes.

Nesting elements

Elements can be placed inside other elements, acting as an emphasis to draw attention to the user..

For example, opening a <p> element first, then opening a <strong> element, to be appropriate we can consider the <strong> element as a level element of the <p> element, so when closing the tag we must close the <strong> element first, and then close the <p> tag. The element "<strong>nóng</strong>" has the effect of bolding the word "nóng" in the text "thời tiết hôm nay thật nóng bức".

Block level element and HTML inline

A block element is any element that begins a new line (e.g. paragraph) and uses the full width of the page or container. The block element can take up a single line or multiple lines and has a line break before and after the element. Easier to understand than block elements are usually structural elements on the page.

For example, a representative block element can represent headers, paragraphs, navigation menus, and more. A block element will not be nested inside an inline element, but block-level elements can be nested.

An inline element is an element inside an element block and is usually used to wrap around a piece of text.

An inline element does not start on a new line.

An inline element only takes up as much width as needed.

What is "div" tag in HTML?

The <div> tag in HTML is an acronym for division, the <div> tag does not affect the content or layout it is usually used to group HTML elements, used to create a block level area for styling with CSS or manipulating scripts.

What is "span" tag in HTML?

The Span tag in HTML is considered an inline element, used to format text or a piece of text. It is similar to div tag but div tag is used for element blocks while span is used for inline elements. It is mainly used whenever users want to group inline elements into the structure of their code.

Span tags are used to style specific assets using the element class or the id attribute. It is not possible to manually make visual changes using the span tag in an HTML document. Using span tags in your code reduces code and HTML attributes.

HTML <img> Tag

The <img> tag is used to embed an image in an HTML page. Images are not technically inserted into a web page; images are linked to web pages. The <img> tag creates a holding space for the referenced image.

The <img> tag has two required attributes:

  • src - Specifies the path to the image

  • alt - Specifies an alternate text for the image, if the image for some reason cannot be displayed

Note: Also, always specify the width and height of an image. If width and height are not specified, the page might flicker while the image loads.

HTML Forms

An HTML form is used to collect user input. The user input is most often sent to a server for processing.

The HTML <form> element is used to create an HTML form for user input. It is a container for different types of input elements, such as: text fields, checkboxes, radio buttons, submit buttons, etc.

The HTML <input> element is the most used form element. An <input> element can be displayed in many ways, depending on the type attribute.

Here are some examples:

Type

Description

<input type="text">

Displays a single-line text input field

<input type="radio">

Displays a radio button (for selecting one of many choices)

<input type="checkbox">

Displays a checkbox (for selecting zero or more of many choices)

<input type="submit">

Displays a submit button (for submitting the form)

<input type="button">

Displays a clickable button

The <button> tag defines a clickable button. Inside a <button> element you can put text (and tags like <i>, <b>, <strong>, <br>, <img>, etc.). That is not possible with a button created with the <input> element!
